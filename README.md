# 냉장고 속 레시피

<p style="color:'#000000'">현대에 들어 요리는 단순히 생존을 위함이 아니라 직업, 취미 등 삶의 중요한 부분을 담당한다. 사람들은 때로 오늘은 무슨 음식을 먹을지, 냉장고에 남은 야채를 어떻게 처리해야 할지 고민한다. 냉장고 속 레시피는 사용자가 가진 요리 재료들로 어떤 음식을 만들 수 있는지를 알려주고 사용자의 선택을 돕는 것을 지향한다.<p>
